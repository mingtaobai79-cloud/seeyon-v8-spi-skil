```xml
<dependency>
    <groupId>com.seeyon</groupId>
    <artifactId>cip-connector-api</artifactId>
    <version>${platform.version}</version>
</dependency>
```

2、实现SPI接口

```
com.seeyon.cip.provider.api.weather.WeatherProviderService
```

###### 3、接口定义

```
public interface WeatherProviderService extends ProviderService {

    @CipCapabilityComment("获取定位")
    LocationDto getLocation();

    @CipCapabilityComment("获取实时天气" +
        "cityCode : 城市编码")
    WeatherData getWeatherCity(String cityCode);

    @CipCapabilityComment("批量缓存获取天气")
    List<WeatherData> getWeatherBatch();

    @CipCapabilityComment("获取能力类型")
    default CapabilityEnum getCapabilityEnum() {
        return CapabilityEnum.Weather;
    }
}
```
