## 1 应用场景

1.1 接入新的对象存储;

1.2 三方文件加密机集成（上传加密、下载解密）

## 2 接口说明

接口名称：com.seeyon.boot.starter.file.spi.StorageSpi

```xml
<dependency>
    <groupId>com.seeyon</groupId>
    <artifactId>boot-starter-file</artifactId>
    <!-- 请以实际环境的依赖版本号为准-->
    <version>3.8.211</version>
</dependency>
```

## 3 接口实现

spi代码仓库获取及工程初始化请参考：
SPI扩展

```java
public class UnifyFileService implements StorageSpi{
    //返回私有桶名称
    @Override
    public String getPrivateBucketName() {return "";}
    //返回公有桶名称
    @Override
    public String getPublicBucketName() {return "";}
    //返回摆渡桶名称
    @Override
    public String getFerryboatBucketName() {return "";}
    //附件上传
    @Override
    public UploadResultDto upload(SpiUploadRequestDto spiRequestDto) {return null;}
    //附件下载
    @Override
    public void download(String storageKey, String bucketName, HttpServletResponse response) {}
    //判断指定桶里面是否存在storageKey
    @Override
    public boolean exist(String storageKey, String bucketName) {return false;}
}
```
