#### 2.5.5 查看组织映射数据

其中名称为该组织数据的名字，源id为该数据在V5系统中的id，目标id为该数据在V8系统中的id，映射类型源值为该数据在V5和V8相同数值作用于匹配的数据，映射类型为用作与映射的字段

只有源id和目标id都有值的数据才能作用于组织映射作用，只有源id没有目标id的数据表明该数据只存在于V5系统并不存在于V8系统
