# 上传下载文件扩展

> Source: https://www.yuque.com/seeyonkk/v8/ibmhcc3391u12fsg

支持版本：5.8 及以上版本

###### 1、使用场景

在系统中对文件的上传下载操作进行拦截，针对文件做处理然后在对文件进行上传下载。比如：我们需要对上传的文件先做敏感词校验，校验通过以后再做上传

###### 2、支持范围

支持全系统的文件上传和下载拦截

###### 3、实现步骤

参考SPI的开发规则：
开发准备

1、引入maven依赖

```xml
<dependency>
  <groupId>com.seeyon</groupId>
  <artifactId>boot-starter-file</artifactId>
  <version>5.8.0</version>
</dependency>
```

2、需要实现的接口

```
com.seeyon.boot.starter.file.spi.StorageInterceptorSpi 
```

3、接口详解

```
public interface StorageInterceptorSpi {
    //文件上传前
    default StorageUploadRequestDto beforeUpload(StorageUploadRequestDto requestDto) {
        return requestDto;
    }
    //文件上传后
    default UploadResultDto afterUpload(StorageUploadRequestDto requestDto, UploadResultDto uploadResult) {
        return uploadResult;
    }
    //文件下载前
    default StorageDownloadRequestDto beforeDownload(StorageDownloadRequestDto requestDto) {
        return requestDto;
    }
    //文件下载后
    default DownloadResultDto afterDownload(StorageDownloadRequestDto requestDto, DownloadResultDto downloadResultDto) {
        return downloadResultDto;
    }
}
```

###### 4、服务重启

对SPI需要生效的服务进行重启
