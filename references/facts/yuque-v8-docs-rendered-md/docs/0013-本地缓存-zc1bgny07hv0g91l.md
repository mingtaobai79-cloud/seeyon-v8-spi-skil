# 本地缓存

> Source: https://www.yuque.com/seeyonkk/v8/zc1bgny07hv0g91l

作者：陈晓东

时间：2025-08-15

提供了基于堆外内存的缓存组件，应用可使用该组件实现数据的缓存，接口方法定义如下：

```java
public class LocalCache<K,V> implements Serializable {

    /**
     * 获取缓存存储的数据长度
     * @return 数据长度
     */
    public long size() {
        return ohCache.size();
    }

    /**
     * 判断缓存是否为空
     * @return 是否为空
     */
    public boolean isEmpty() {
        return ohCache.size() == 0;
    }

    /**
     * 判断缓存是否包含指定key
     * @param key key值
     * @return 是否包含指定key
     */
    public boolean containsKey(K key) {
        return ohCache.containsKey(key);
    }

    /**
     * 获取指定key的值
     * @param key key值
     * @return 指定key的值，如果没有返回null
     */
    public V get(K key) {
        return ohCache.get(key);
    }
```

通过LocalCacheBuilder初始化LocalCache对象后可以针对localCache进行读写操作。示例如下：

```
LocalCache<Long, User> localCache = LocalCacheBuilder.defaultBuilder().build();
User user = new User();
// 此处对user属性进行设置，或者是从数据库中获取的user
localCache.put(user.getId(), user);
```

在具体使用的时候可以通过LocalCacheBuilder对缓存的大小、过期时间以及过期策略进行设置，示例如下：

```
LocalCache<Long, User> localCache = LocalCacheBuilder.defaultBuilder()
.capacity(1024*64) // byte大小
.loadFactor(0.75f) // 加载因子
.ttlMillis(60000L * 30) // ttl过期时间（毫秒）
.eviction(CacheEviction.LRU) // 驱逐算法 LRU/NONE
.build();
User user = new User();
// 此处对user属性进行设置，或者是从数据库中获取的user
localCache.put(user.getId(), user);
```
