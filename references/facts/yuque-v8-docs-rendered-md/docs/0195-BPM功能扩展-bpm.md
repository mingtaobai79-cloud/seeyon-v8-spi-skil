##### 4.2 与V5事件的区别

例如提交前操作：

V5是可以实现调用后端接口，前端弹框进行提示，或者让用户选择是否继续提交。

V8只是调用后端接口，但是这个扩展口子没有办法进行前端弹框让用户选择是否继续提交。

要实现V5的校验弹框，只能通过按钮绑定点击事件执行页面规则，在页面规则中调用后端微流程，执行逻辑处理，然后根据返回值，在前端页面进行弹框提示。
