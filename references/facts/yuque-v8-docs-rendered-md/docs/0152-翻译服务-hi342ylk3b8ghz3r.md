###### 1、使用场景

提供中、英、法、德等常用的多国语言文本翻译能力，支持语种识别

###### 2、操作步骤

SPI开发规则，参考：
开发准备

1、SPI工程中加入maven依赖

```xml
<dependency>
    <groupId>com.seeyon</groupId>
    <artifactId>cip-connector-api</artifactId>
    <version>${platform.version}</version>
</dependency>
```

2、实现SPI接口

```
com.seeyon.cip.provider.api.translation.TranslationProviderService
```

###### 3、接口定义

```
public interface TranslationProviderService extends ProviderService {
    @CipCapabilityComment("翻译原文为哪个语种" +
        "query : 翻译原文" +
        "to : 翻译成哪个语种" +
        "appName : 应用名称")
    String translate(String query, String to,String appName);

    @CipCapabilityComment("获取语种列表")
    List<LanguageDto> getLanguages();

    @CipCapabilityComment("获取能力类型")
    default CapabilityEnum getCapabilityEnum() {
        return CapabilityEnum.Translation;
    }
}
```
