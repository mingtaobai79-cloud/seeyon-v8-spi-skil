# 概述

> Source: https://www.yuque.com/seeyonkk/v8/ozqwzy676zflehr4

作者：上官端阳

最后更新时间：2025-09-09

## 定位：数据迁移工具

通过在A9平台中搭建UDC迁移应用，提供标准数据迁移能力，实现源系统业务数据快速迁移至A9协同运营平台。

## 核心特性

1、表单、公文、文档、新闻公告、自由协同数据迁移

2、表单、公文实现根据源系统自动生成实体、文单

3、组织自动映射

4、文件支持上传、挂载双机制

5、支持错误数据回滚重新迁移

## 适用范围

○
源系统：致远V5协同运营平台

○
V8版本要求：V5.3以上
