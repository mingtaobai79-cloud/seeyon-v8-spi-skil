## 1. 应用场景

### 1.1 场景一：监听消息推送至企微

致远内网平台作为消息中心，‌是一个中介系统，用于在不同的模块或服务之间传递消息。将目前接收到的第三方待办和消息或平台消息统一以消息的形式推送至企微，企微中以消息卡片的形式进行展示。同时用户可点击消息卡片进行平台页面或第三方页面跳转。

#### 1.1.2 参数配置

##### 1
V8域名配置

进入NACOS，在public 配置如下参数

```
seeyon:
  system:
    domain: dev.seeyonv8.com
```

##### 2
配置spi插件

进入NACOS，在cip-connector微服务下配置如下参数

```
seeyon:
  spi:
    enable: true
    spi-plugins:
      - maven:com.seeyon,cip-mobile-plugin-weixin,1.0-SNAPSHOT
```
