# 自定义门户元素

> Source: https://www.yuque.com/seeyonkk/v8/vzfhydi7b8rtgf9e

适用版本：5.1以上版本

作者：杨颜华
最后更新：2025-05-24

## 1. 应用场景

标品的门户模板里面的元素区域不满足客户场景 ，可以新增一个自定义的元素组件替换标准元素组件 或者新增一行文字等内容元素区域展示

场景一:  在元素区域内容自定义

场景二：在元素区域替换标准菜单元素
