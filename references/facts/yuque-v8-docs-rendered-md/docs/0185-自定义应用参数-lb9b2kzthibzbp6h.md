## 4.注意事项

设计态参数有过修改、删除操作，在原样多次导入应用到其他系统，遇到参数不准确、混乱时，可以将运行态的

ctp_parameter_definition表对应的参数数据删除。然后在重新发布，参数就会是最新的。
